# Squash and Stretch：让动画活起来的小秘密

Josh 这篇文章的好处，在于它没有把动画讲成“参数调优”，而是把传统动画原理重新翻译成前端开发能直接理解的语言。Squash and Stretch 这种技巧一旦用对，界面会立刻多出一层生命感。

对做交互和品牌体验的人来说，这类内容的价值往往远高于某个具体库的 API 教程。
