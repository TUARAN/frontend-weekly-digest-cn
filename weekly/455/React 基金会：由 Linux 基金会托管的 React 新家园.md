# React 基金会：由 Linux 基金会托管的 React 新家园[](https://react.dev/blog/2026/02/24/the-react-foundation#undefined)

2026 年 2 月 24 日，作者：[Matt Carroll](https://x.com/mattcarrollcode)

![](https://react.dev/images/blog/react-foundation/react_foundation_logo.webp)
![](https://react.dev/images/blog/react-foundation/react_foundation_logo_dark.webp)

React 基金会已正式成立，并由 Linux 基金会托管。

[在 10 月](https://react.dev/blog/2025/10/07/introducing-the-react-foundation)，我们宣布了成立 React 基金会的意向。今天，我们很高兴与大家分享：React 基金会已正式启动。

React、React Native，以及 JSX 等支持性项目不再归 Meta 所有——它们现在归 React 基金会所有。React 基金会是一个由 Linux 基金会托管的独立基金会。你可以在 [Linux 基金会的新闻稿](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-react-foundation) 中阅读更多内容。

### 创始成员 [](https://react.dev/blog/2026/02/24/the-react-foundation#founding-members)

React 基金会有八家白金级创始成员：**Amazon**、**Callstack**、**Expo**、**Huawei**、**Meta**、**Microsoft**、**Software Mansion** 和 **Vercel**。**Huawei** 在我们 [10 月的公告](https://react.dev/blog/2025/10/07/introducing-the-react-foundation) 之后加入。React 基金会将由一组董事会进行治理，董事会由每位成员的代表组成，并由 [Seth Webster](https://sethwebster.com/) 担任执行董事。

![](https://react.dev/images/blog/react-foundation/react_foundation_member_logos_updated.webp)
![](https://react.dev/images/blog/react-foundation/react_foundation_member_logos_dark_updated.webp)

### 新的临时领导委员会 [](https://react.dev/blog/2026/02/24/the-react-foundation#new-provisional-leadership-council)

React 的技术治理将始终独立于 React 基金会董事会——React 的技术方向将继续由那些为 React 做出贡献并维护它的人来决定。我们已成立一个临时领导委员会来确定这一治理结构。我们将在未来几个月分享最新进展。

### 下一步 [](https://react.dev/blog/2026/02/24/the-react-foundation#next-steps)

要完成这次过渡仍有一些工作要做。在未来几个月里，我们将：

- 完成 React 的技术治理结构
- 将代码仓库、网站以及其他基础设施迁移到 React 基金会名下
- 探索用于支持 React 生态系统的相关项目计划
- 启动下一届 React Conf 的筹备工作

随着这项工作推进，我们会持续分享更新。

### 致谢 [](https://react.dev/blog/2026/02/24/the-react-foundation#thank-you)

如果没有过去十年间塑造了 React 的数千名贡献者，这一切都不可能实现。感谢我们的创始成员，感谢每一位提交过 Pull Request、提出过 Issue、或帮助他人学习 React 的贡献者，也感谢每天使用 React 进行构建的数百万开发者。React 基金会因这个社区而存在，我们期待与大家一同构建它的未来。
