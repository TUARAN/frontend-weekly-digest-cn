# 两个圆、一个箭头与 Anchor Positioning

最能说明一个新 CSS 能力价值的，往往不是宏大宣言，而是一个足够具体的小例子。两个圆、一个箭头，这种看起来简单的界面关系，过去常常要靠绝对定位、手算偏移甚至 JavaScript 修正来维持。一旦布局稍微变化，脆弱性就暴露出来。

Anchor Positioning 值得期待的地方，正是在于它试图把“元素之间的几何依赖关系”变成平台可表达的事情。很多前端布局痛点，说到底都不是不会摆，而是缺乏一种稳定描述“我应该相对谁出现”的原生方式。

这类文章很好，因为它不抽象。它让你直观看到一个新特性到底替你省掉了哪些历史包袱，也更容易判断它什么时候值得在项目里引入。
