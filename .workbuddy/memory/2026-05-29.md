## 添加 syncblog.cn 同步功能

为前端周刊站点添加了「同步到 syncblog」功能，支持将周刊内容一键传送到 syncblog.cn 编辑器。

### 技术方案
syncblog.cn 使用跨窗口 postMessage 通信（非 REST API）：
1. `window.open('https://syncblog.cn/md/#content-sync')` 打开编辑器
2. syncblog 发送 `SYNCBLOG_IMPORT_READY` 信号
3. 通过 `popup.postMessage()` 发送文章 payload
4. 格式：`{ type: 'SYNCBLOG_IMPORT_ARTICLE', title, markdown, canonicalUrl, tags }`
