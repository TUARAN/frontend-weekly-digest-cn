# 🧭 Frontend Weekly Digest 中文版

> 📘 将全球前端领域的高质量内容带给中文开发者

本项目是对英文原版《Frontend Weekly Digest》的持续翻译整理。内容涵盖 Web 开发、CSS、JavaScript、React、开发工具、性能优化、动效设计等多个方向。每期整理约 20~30 篇精选内容，力求信息密度高、可读性强、技术趋势清晰。

---

## 📖 项目背景

**我们之前：**

之前在 [Flowus](https://flowus.cn/) 平台上，我和另外 4 位小伙伴一起逐期翻译《前端周刊》内的每篇文章。由于 Flowus 付费空间有限，现在计划将项目迁移到 GitHub，同时号召更多的小伙伴加入进来，一起用热爱发电！

**我们发现：**

**中文社区缺乏系统性跟踪海外一手前端动态的内容**。而国外像 Medium、Smashing、LogRocket、WebKit Blog 等一线平台，每周都在迸发新洞见、新 API、新标准、新框架。**希望你读到的不只是文章本身，更是一种与全球前端开发者同步成长的方式。**

我们不靠算法推荐，而是靠前端开发者对技术趋势的敏锐、对知识分享的热爱，手动**为你筛选每周全球社区中最新、最热、最值得关注的前端文章**。
