# 如何在 React 应用中测试多语言无障碍

这篇文章抓住了一个很容易被忽略的事实：无障碍测试如果只在单一语言环境里做，很多真实问题根本不会暴露。文本长度、语义差异、阅读顺序和朗读体验，都会随着 locale 改变。

因此，把多语言和可访问性放在一起验证，才更接近真实产品质量，而不是纸面合规。
