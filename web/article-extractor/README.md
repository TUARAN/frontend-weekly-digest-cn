# Article Extractor

一个自动的文章提取工具，主要流程如下：

1. 上传一个需要分析的 md 文件，程序会自动分析出其中的文章链接。
2. 手动对生成文章进行确认，主要是去除原网页中的一些不需要的内容，比如广告、评论等。
3. 抓取内容后，程序会自动生成一个新的 md 文件，里面主要包含预设的提示词，以及抓取到的文章内容，用户可以在这个 md 文件中对内容进行修改和润色。
4. 用户确认 md 文件内容后，可以直接复制到剪贴板，然后交由其他工具进行后续的处理，不做自动化翻译处理主要是基于用户对使用的工具不一致进行考虑，用户看可以选择自己喜欢的 AI 翻译工具进行翻译处理。


# 使用说明

终端定位到当前工程目录，首先安装依赖：

```bash
npm install

cd ./client
npm install
```

安装完成后，回到根目录，执行以下命令：

```bash
npm run dev
```

dev 命令会同时启动后端和前端的开发环境，主要执行以下两个命令：

```bash
npm run dev:server
npm run dev:client
```

前端没有自动打开浏览器，所以需要手动打开浏览器访问 http://localhost:5173/ 进行使用。


# 修改附加的 prompt

如果需要修改附加的 prompt，可以在`./server/prompts/translate_prompt.md`文件中进行修改

**注意：** 由于代码中使用的是硬编码替换`[${原文英文标题}](${原文链接})`这个内容，所以这里的 prompt 中需要保留这个内容，修改时请注意不要删除或者修改这个内容，否则会导致生成的 md 文件中无法正确替换原文标题和链接。
