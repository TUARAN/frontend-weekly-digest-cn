# Astro vs Next.js：内容型网站里 SSG 何时胜过 React

这类对比文最有价值的时候，不是它告诉你谁赢，而是它逼你先问清自己的网站到底是什么类型。内容站、营销站、文档站和产品应用，本来就不该共享同一种默认架构。

如果你的站点交互很轻、内容很重，那 SSG 往往比全量 React 更自然，这正是 Astro 持续有吸引力的原因。
