# AI 生成的 UI 默认并不无障碍

这篇文章切得很准：当大家开始习惯让模型快速生成界面时，最容易被一起“自动生成”的，其实是可访问性债务。视觉上能跑起来，不代表键盘可操作、语义结构合理、状态反馈完整，更不代表真实用户能顺畅使用。

它的提醒很直接，也很现实。AI 生成 UI 现在已经足够快，但质量门槛并没有被自动跨过去。对前端团队来说，这意味着生成式开发流程里必须把无障碍检查、语义审查和交互验证放进默认步骤，否则只是把旧问题更快地批量制造出来。
