# State of JavaScript 2025：TypeScript 统治力继续巩固

如果说前些年我们还在讨论 TypeScript 到底是不是“值得”，那现在问题基本已经变成了：你准备在什么边界上不使用它。State of JavaScript 2025 的意义，正在于它再次确认了一件事，TypeScript 已经从偏好选择走向了生态默认。

这并不意味着 JavaScript 生态失去活力，反而说明它在成熟。一个成熟生态的特征，往往不是不断出现完全替代旧范式的新东西，而是逐步形成一组足够稳固的共识。TypeScript 的统治力继续巩固，本质上就是这种共识在扩大。

对团队来说，这类调查最大的价值不是“跟风”，而是帮助判断哪些技术决策已经从实验区进入了基础设施区。一旦进入这个阶段，招聘、协作、代码治理和工具链都会随之改变。
