# Cloudflare：Agent Readiness 分数来了，你的网站准备好给 Agent 用了吗？

这篇文章最值得注意的，不是 Cloudflare 又做了一个“评分工具”，而是它把一个新问题公开摆上了台面：网站过去主要面向用户和搜索引擎，现在还得考虑能否被 Agent 正确理解、导航和调用。

所谓 Agent Readiness，本质上是在问：你的页面结构、元数据、内容暴露方式和交互路径，是否足够清晰，能让模型或代理系统稳定消费。这个问题后面大概率会越来越常见。
