本月 Astro 生态依旧热闹非凡：从核心产品更新、社区活动到第三方工具与主题模板，生态的每个角落都在不断进化。下面按板块为你梳理 2025 年 11 月 Astro 的重要动态。

## 🚀 Mission Control 更新

- 🎇 **Stainless 推出 Astro 驱动的文档平台**：Stainless 发布了全新的文档平台，底层由 Astro 提供驱动，展示了 Astro 在文档类站点上的性能与可维护性优势。

- ⚛ **参与 React 生态调研**：如果你在项目中同时使用 Astro + React，可以在 State of React 2025 问卷中分享你的实践体验。

## 🛠 Tips & Tools：社区最新工具与集成

以下是本月出现的一些 Astro 相关工具与集成，帮助你更高效地构建项目：

- **Astro Web Vitals**：轻量级 Web Vitals 监控组件，支持调试浮层与分析集成  
- **Astro Capo**：在 Astro 中通过 capo.js 自动整理 `<head>` 内容  
- **Astro i18nya**：面向 Astro 的多语言（i18n）支持方案  
- **Astro AI Coauthor**：面向开发者的 AI 文档助手，帮助撰写与维护文档  
- **Astro Cloudflare Pages Static**：在 Cloudflare Pages 上支持客户端动态路由的集成  
- **Astro Directify**：用于服务器端声明式控制流的指令集成  
- **Astro WhatsApp**：为站点添加固定悬浮 WhatsApp 图标的组件  
- **Mona Kiosk**：基于 Polar.sh 的内容变现集成  
- **Zephyr Astro**：面向 Zephyr Cloud 部署与分析平台的 Astro 集成  
- **Astro Orbit**：包含设计系统、可复用组件与 Tailwind CSS v4 集成的主题包  
- **Astro Snapshot**：在构建时自动为页面生成截图  
- **Silk Astro**：为 Silk（零运行时 CSS-in-TypeScript）提供的 Astro 集成，支持 islands 架构与局部水合  
- **Scalar API Reference for Astro**：用于生成 API 文档的 Scalar 组件集成  
- **EdgeOne Astro**：将 Astro 项目部署到 EdgeOne Pages 的适配器  
- **Astro Auth**：基于 Auth.js 的 Astro 认证集成，提供多种登录方式、会话管理与自然融入 Astro 的 UI 组件  
- **Astro Dynamic Assets**：基于 React 帮助你生成自定义图片（如 Open Graph 图、社交媒体配图等）  
- **Astro Trakt.tv Loader**：面向 Astro 内容层的 Trakt.tv 内容加载器  
- **Astro HTML Minifier Next**：在构建时自动压缩 HTML 资源的集成  
- **AstroJS Atom**：在 Astro 项目中生成 Atom RSS Feed 的库  
- **Vitest Browser Astro**：在真实浏览器环境中使用 Vitest 测试 Astro 组件  

此外，还有一些**由 Astro 构建的项目**值得一看：

- **Git Views**：为 GitHub 个人主页与仓库添加浏览量统计  
- **Release Image Generator**：生成带自定义图案、颜色与文字的随机图片  
- **Starlight Detector**：检测网站是否使用 Starlight 构建（类似 isAstro，但功能稍弱）
