# 更多网站支持自动进入视频画中画

自动进入视频画中画，看上去像一个很细碎的小更新，但它其实非常体现浏览器正在重新理解真实使用场景。用户看视频的时候，并不会总老老实实停留在当前标签页。切走、查资料、回消息、顺手做别的事，才是更常见的行为。

如果浏览器能更智能地在合适时机维持视频连续性，体验提升会非常直接。它减少的不是某个点击动作，而是任务切换时的上下文中断。很多产品优化，真正有价值的地方并不在于功能本身多复杂，而在于它是否顺着人的真实行为模式去设计。

这类能力也提醒我们，现代 Web 平台已经不只是“把内容显示出来”，而是在越来越多地参与多任务交互和系统级体验编排。浏览器如果持续把这些场景接住，前端产品的很多细节成本都会下降。
