# 三月 Web 平台新动态

这种月度汇总类文章的价值，不在于替代深读，而在于帮你迅速建立“平台最近到底在往哪儿走”的整体感觉。单看某一个 API 更新，容易觉得只是零散新闻；把一整个月的变化放在一起，趋势就会清晰很多。

三月这一波更新最值得注意的，是平台能力继续往“更细、更稳、更实用”的方向推进。不是那种一下子颠覆开发模式的大爆点，而是持续把开发者过去要靠工具、约定和补丁解决的问题收回到平台层。真正决定前端工程质量的，往往就是这种长期的小步快跑。

所以这类文章最适合拿来做“方向校准”。如果你最近一段时间没怎么跟浏览器动态，读一次这种总结，基本就能把自己的认知重新拉回到当前语境里。
