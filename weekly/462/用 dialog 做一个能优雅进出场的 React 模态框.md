# 用 dialog 做一个能优雅进出场的 React 模态框

`<dialog>` 这几年越来越值得重新看待，因为它让很多模态框的基础能力回到了平台原生层。但真正难的从来不是“能弹出来”，而是进场、退场、焦点管理和交互细节如何一起做好。

这篇文章的价值就在于它没有停在 API 层，而是往真实可用组件迈了一步。
