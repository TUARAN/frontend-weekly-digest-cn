## 今日精选

### 1. Claude Opus 4.8 发布：代码漏检率降 75%，Agent 判断力全面升级

**分类**：AI Coding · **来源**：Anthropic Newsroom

Anthropic 发布 Opus 4.8，Online-Mind2Web 评测得分 84%，代码错误漏检率降低约 75%，诚实度与对齐表现同步提升。2.5 倍速模式价格降至前代的 1/3，已登陆 Claude Code 等全平台。

> Opus 4.8 将 Agent 任务判断可靠性推向新高度，AI 编程从"辅助"进化为"可信赖的协作者"

[阅读原文](https://www.anthropic.com/news/claude-opus-4-8)

### 2. Claude Code 推出"动态工作流"，单会话并行调度数百子 Agent

**分类**：AI Coding · **来源**：Claude Blog

Claude Code 新增动态工作流功能，通过动态编写脚本在单次会话中并行运行数十至数百个子 Agent，结果验证后呈现。已用于跨代码库 bug 查找、Bun 从 Zig 到 Rust 大规模语言移植等复杂任务。

> 多 Agent 协作模式让 Claude Code 可处理超大规模项目，AI 编程正式进入"全员并行"时代

[阅读原文](https://claude.com/blog/introducing-dynamic-workflows-in-claude-code)

### 3. 阶跃星辰开源 Step 3.7 Flash：198B MoE，兼容 Claude Code / MCP

**分类**：AI Coding · **来源**：X @StepFun_ai

阶跃星辰发布开源模型 Step 3.7 Flash，198B 参数 MoE（11B 活跃），ClawEval-1.1 得分 67.1 分位列第一。兼容 Claude Code、MCP 协议，支持 Mac Studio M4 Max 本地运行，Apache 2.0 许可开源。

> 首个兼容 Claude Code 的开源 Agent 模型，本地部署能力对前端开发者极具吸引力

### 5. Anthropic 完成 650 亿美元 H 轮融资，年化收入突破 470 亿美元
