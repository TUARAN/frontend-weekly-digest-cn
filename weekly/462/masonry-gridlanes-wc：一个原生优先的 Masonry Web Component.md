# masonry-gridlanes-wc：一个原生优先的 Masonry Web Component

这篇文章的价值，在于它不是单纯展示一个瀑布流组件，而是在强调“原生优先”的实现思路。也就是说，优先利用平台已有能力，不够的部分再用脚本补，而不是一开始就把整套布局逻辑全扛到 JavaScript 里。

这种方向对 Web Components 很重要。因为组件的长期价值，往往不只是功能有没有做出来，而是它跟平台能力之间的耦合程度够不够健康。
